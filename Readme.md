## 🎯 About The Project

OverClocked is a dedicated online marketplace for computer hardware enthusiasts, featuring a **three-tier role-based architecture**:
- **👤 Customer**: Browse components, compare technical specs, and complete secure purchases.
- **🏪 Seller**: Manage hardware listings, track inventory, process orders, and view sales analytics.
- **🛡️ Admin**: Oversee the entire catalog, approve new sellers, and manage users.

---

## ✨ Key Features

- **Advanced Hardware Discovery**: Detailed specs sheets, deep brand filtering, and real-time smart cart updates.
- **Role-Based Dashboards**: Custom analytics, inventory controls, and order management for Sellers and Admins.
- **Secure Ecosystem**: JWT-based authentication, role-based middleware, and secure Razorpay payment integration.
- **Modern Frontend**: Built with React 19, Tailwind CSS, Redux Toolkit, shadcn/ui, and Framer Motion animations.
- **Robust Backend**: Fast Express.js API, MongoDB aggregations, Cloudinary image storage, and Nodemailer.

## 🚀 Quick Start

### Prerequisites
- [Node.js](https://nodejs.org/) (v18+)
- [MongoDB Atlas](https://www.mongodb.com/cloud/atlas) or local MongoDB instance
- [Git](https://git-scm.com/)

### Installation

```bash
# Clone the repository
git clone https://github.com/Mann275/E-commerce.git
cd E-commerce
```

**1. Backend Setup**
```bash
cd server
npm install
# Create your .env file here based on requirements
npm run dev
# Server will start on http://localhost:8000
```

**2. Frontend Setup**
```bash
cd ../client
npm install
# Create your .env file here based on requirements
npm run dev
# Frontend will start on http://localhost:5173
```
